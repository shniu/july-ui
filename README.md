
# TinyURL

## 项目简介

TinyURL 项目是一个短网址服务，主要功能是将长网址转成短网址，方便在社交网络上传播。

从需求上来讲，粗略来看是一个很简单的项目，但是在不同用户场景下，却不是那么简单。做这个项目的初衷是为了找一个简单的业务来练习构建一个高并发高性能的SAAS服务，其中需要将微服务、分布式、云原生、CI/CD等技术应用进去；从这个层面来讲，我要构建的是一个能应对百万级并发甚至更高并发的系统，而不是一个玩偶项目。
